Das Aktivitätsdiagramm zu UC7 „Putzplan erstellen“ beschreibt den grundlegenden Ablauf zur Erstellung eines neuen Putzplans innerhalb der WG. Zunächst startet ein Benutzer die Funktion zur Putzplanerstellung und legt einen Zeitraum fest, für den der Plan gelten soll. Das System prüft anschließend, ob dieser Zeitraum gültig ist. Ist dies der Fall, können die anstehenden Aufgaben definiert und verfügbaren Bewohnern zugewiesen werden. Danach wird der Putzplan gespeichert und für die Mitglieder sichtbar gemacht. Tritt hingegen ein Fehler auf, etwa durch einen ungültigen Zeitraum oder fehlende verfügbare Bewohner, beendet das System den Vorgang mit einem entsprechenden Hinweis. Das Diagramm zeigt damit den Kernprozess der Funktion F02 Putzplan, nämlich die strukturierte Erstellung, Zuweisung und Bereitstellung von Reinigungsaufgaben für WG-Mitglieder.
